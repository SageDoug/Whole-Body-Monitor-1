# Whole Body Monitoring Web App

## Features
- Steps
- Calories
- Active Minutes
- Heart Rate (Real-time)
- ECG
- SpO2
- Menstrual Cycle Tracking
- Stress Levels
- Body Composition
- Sleep Stages
- Sleep Apnea
- Blood Pressure
- Energy Score
- Antioxidant Index
- Fall Detection (Real-time alert)

## Setup

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev

## Environment Variables

Create a .env file in server:

MONGO_URI=your_mongodb_connection_string
